# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| v0.3.x (devnet) | ✅ Active development |
| < v0.3.0 | ❌ No support |

## Reporting a Vulnerability

**Do not open a public issue.**

Email: **gorkhmazb23@gmail.com**
Telegram: **@unc_gorkh**

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

### Response timeline
- **Acknowledgment:** Within 48 hours
- **Initial assessment:** Within 1 week
- **Fix timeline:** Depends on severity

### Scope
- Solana programs (human_registry, agent_registry, delegation, receipts)
- TypeScript SDK
- KYC Issuer service
- Key management and access control

### Out of scope
- Veriff's infrastructure
- General Solana network issues
- Social engineering attacks

## Bug Bounty
No formal bug bounty program at this time. Significant findings may be rewarded at our discretion.
